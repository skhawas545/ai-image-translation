How to Use the App

To use the AI-Based Picture Translation App, follow these steps:

  • Upload an image with text using the "Upload Image" button.
  • Select the language you want the text to be translated into.
  • Click "Translate Image" to see the translated text.
  • You can download the translated text or image for later use.

Common Issues

If you are experiencing problems, try the following:

  • Make sure the image you upload is clear and readable.
  • Ensure that the file size is within the allowed limit.
  • If translation fails, try a different image or language selection.
