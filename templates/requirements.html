Functional Requirements

Requirement ID Description Priority
FR1 Users should be able to upload images containing text. High
FR2 App should recognize text in the uploaded image using OCR. High
FR3 App should detect the source language of the recognized text. High
FR4 App should translate the detected text into the target language selected by the user. High
FR5 Translated text should be displayed in real-time on the image or as plain text. Medium
FR6 Users should be able to download the translated text or the image. Low
